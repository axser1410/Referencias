Termux


Instalacion de Termux

Para instalar termux en un android es necesario instalarlo desde F-droid (no instalarlo desde la Play Store). Para instalarlo se debe dar los permisos correspondientes, y ya en la terminal de Termux se ejecuta los siguientes comandos:

apt update: se utiliza para actualizar al lista de paquetes instalados.

apt upgrade: se utiliza para actualizar los paquetes instalados


Instalacion de la API de Termux

se puede verificar las herramientas de termux escribiendo en la terminal termux y dando doble tabulador

para instalar la api de termux se ejecuta el comando pkg install termux-api y para que funcionen las herramientas instaladas de la API de Termux es necesario instalar API de Termux

control remoto con ssh

instalacion del paquete con : "pkg install openssh"

despues de instalarlo para iniciar el ssh es recomendable agregar una contraseña con el comando: "passwd" (si no permite ejecutar el comando, se soluciona actualizando las librerias y paquetes con "apt update" y "apt upgrade" posteriormente se vuelve a ejecutar "passwd")

se ejecuta el comando sshd para iniciar el servidor ssh.(en caso de que no permita ejecutar el comando ejecutamos el comando "ssh-keygen -A" para generar las claves de los hosts)

para conectar desde la powerShell se utiliza el comando "ssh usuario@ip -p 8022" utilizando despues la clave que se genero aqui

lista de comandos ejecutados en el Termux

  • whoami(ver el usuario del equipo)
  • ifconfig(ver el ip del equipo)

Bootstrap

¿QUE ES BOOTSTRAP?

Bootstrap es un framework que permite a los desarrolladores web construir páginas web responsives de una forma más rápida y sencilla. En este sentido, proporciona un conjunto de componentes y plantillas CSS, HTML y JavaScript que cualquiera puede utilizar o modificar de manera gratuita.

INSTALACION

Existen 2 opciones de instalacion o de agregar a los proyectos:
1. Agregar las siguientes linaes de codigo. La referencia se encuentra en la pagina oficila de BOOTSTRAP

agregar lo siguiente en como un link de CSS
